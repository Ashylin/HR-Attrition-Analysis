# 🏢 HR Attrition Analysis
## Predicting Employee Attrition Using Machine Learning

### 📌 Project Overview
This project analyzes IBM HR data of 1470 employees to identify 
key factors driving employee attrition and predict which employees 
are at risk of leaving.

### 📊 Dataset
- Source: IBM HR Analytics Dataset (Kaggle)
- 1470 employees, 35 features

### 🔍 What I Did
- Data cleaning and preprocessing
- Exploratory Data Analysis (10 charts)
- Feature Engineering
- Random Forest ML Model (85-90% accuracy)
- Business Insights Report

### 🏆 Key Findings
- 16% attrition rate overall
- Low salary is #1 reason employees leave
- Sales department has highest attrition
- Overtime workers leave significantly more
- Younger employees (25-35) leave more

### 🛠️ Tools Used

- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook
